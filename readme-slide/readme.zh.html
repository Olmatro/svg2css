  Inkscape Plugin，能夠將 SVG 轉換為 HTML+CSS。
  2.0.0-alpha-1
  如何安裝
  解壓縮後，把下面這三個檔案放到 /path/to/inkscape/share/extensions
  * svg2css.py * svg.py * svg2css.inx
  如何使用？
  步驟1. 使用"建立圓形、橢圓形與弧形"來畫出圖形。
  步驟2. 存檔為 "HTML+CSS(.html)" 格式。
  注意：你不能用"繪製貝茲曲線及直線"，svg2css 無法轉換它們。
