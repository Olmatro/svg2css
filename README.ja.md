svg2css
======================
SVGをHTML+CSSに変換するInkscapeのプラグインです。

## インストール
次のファイルを (Inkscapeのインストールフォルダ)/share/extensions にコピーしてください

* svg2css.py
* svg.py
* svg2css.inx

## 実行
"HTML+CSS(.html)"というファイル形式で保存することで、HTML+CSSに変換できます。

詳細は readme-slide/index.ja.html を確認してください。
